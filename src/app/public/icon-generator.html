🎨 TwoBeOne Icon Generator

Generate PWA icons for all platforms

📋 How to Use

  1. Click "Generate All Icons" below (auto-generates on page load)
  2. Click "Download All as ZIP" to save all icons at once
  3. Extract the ZIP file to get all 10 PNG icons
  4. Upload all PNG files to /public/icons/ directory in your project
  5. Test by adding the app to your phone's home screen!
