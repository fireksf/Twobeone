This folder will contain your PWA icons.

Please generate icons using one of these methods:

1. Use the icon generator: /scripts/generate-pwa-icons.html
2. Use online tool: https://www.pwabuilder.com/imageGenerator
3. Create manually in design software

Required icons:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png (CRITICAL)
- icon-384x384.png
- icon-512x512.png (CRITICAL)

See /GENERATE_ICONS_INSTRUCTIONS.md for detailed help.
