
  # Twobeone
  #Grow Together
  #In Faith & Love
TwoBeOne helps Christian couples strengthen their relationship through daily devotionals, shared prayers, and meaningful conversations rooted in Scripture.
  This is a code bundle for Twobeone. The original project is available at https://www.figma.com/design/OiNbjfzqcqG6JIzbu8D3Zj/Twobeone.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
